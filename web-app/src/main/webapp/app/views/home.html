யாருக்கு / என்ன பயன் ?

ஒரு சோதனைத்தளம், கடைநிலைப்பயனுக்கல்ல

ஒரு குறிப்பிட்ட கடைநிலைப்பயன்பாட்டிற்காகமட்டுமே உருவாக்கப்படும் மென்பொருளல்ல. பலவிதமான பயன்பாடுகளைக்கொண்ட தமிழ்மென்பொருட்களை இதன்மேல் பிற்காலத்திலுருவாக்க இதுவொரு தளமாக அமையவேண்டுமென்பதே இதனுடைய அடிப்படைநோக்கம். எனவே, இந்தச்செயலியில் காணப்படும் பக்கங்கள் கடைநிலைப்பயன்பாட்டாளர்களுக்கானவையல்ல. இந்தச்செயலியை அதனுடைய நோக்கத்தை எட்டவைக்க உதவக்கூடிய தமிழார்வலர்களுக்கானவை அவை.

சில பக்கங்கள் (எ.கா தமிழ்த்திருத்தி ) போன்றவை கடைநிலைப்பயன்பாட்டைக்கொடுத்தாலும் அந்தத்திருத்தி ஒரு சோதனைத்தளமாகவே [test-bed or a reference implementation] பயன்படுகிறது. ஒரு வலுவான தமிழ்த்திருத்தியை செய்யவிரும்புபவர்கள் தாராளமாக செய்யலாம். இந்தத்தளம் [platform] அதற்கு தேவையான REST APIs களைக்கொடுக்கும். ஓரழகான மற்றும் வலுவான தமிழ்த்திருத்தியை இத்தளங்கொடுக்குஞ்சேவைகளைப்பயன்படுத்தி உருவாக்க ஆவல்கொண்டவர்கள்
