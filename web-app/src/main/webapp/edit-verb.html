<-பின்னுக்குச்செல்க

இவ்வினையடியில் மாற்றஞ்செய்க.

 
வினையடிகளிலிருந்து பொதுவாக வினைமுற்று கீழ்க்கண்ட வாய்ப்பாட்டை பயன்படுத்தி உருவாகும்.
வினையடி + [சந்தி] + காலவிடைநிலை + [சாரியை] + பால்விகுதி = வினைமுற்று.

உருவாக்கப்பட்டுள்ள வினைமுற்றுகள் சரியாவென்று பார்க்கவும். அவற்றில் பிழையிருந்தால்,
    அதைத்தெரியப்படுத்த,
 "இந்த வினைமுற்றுகளில் பிழையிருக்கிறது"
 என்னும் கீழேயுள்ள கட்டத்தில் உங்களின் தேர்வையிடுங்கள்.
 அல்லது அதை சரிசெய்ய, உங்களின்முயற்சியை கீழ்க்கண்டவரிசையிலமைக்கலாம். 

1.முதலில் வேகமானத்தேர்ந்தெடுப்பை முயற்சிக்கவும். நிற்க: ஏற்கனவேயிருக்கும் வினைப்பகுதிகளின் தேர்வை நீக்கிவிட்டு இதை முயற்சிப்பது நல்லது.

2. குறிப்பு:வினையடிகள்   செயப்படுபொருளைகொண்டிருக்கும் (எ.கா கொல் -"பாம்பை" கொன்றேன்).
   சிலவை கொண்டிருக்காது (எ.கா வா - வந்தேன்) .  சிலவை இரண்டுபயன்பாடுகளிலும் வரும் (எ.கா நனை - நனைந்தேன்/நனைத்தேன்).
   அதை சரியாகக்தீர்மானித்து

     செயப்படுபொருளைக்கொண்ட வினையா -  ஆம்/இல்லை
     செயப்படுபொருளிலாத  வினையா     -  ஆம்/இல்லை

ஆகிய இருகேள்விகளுக்கும்  சரியாக முதலில் பதிலளியுங்கள். பின்னர், மற்றமாற்றங்களை ஒவ்வொன்றாகச்செய்யவும்.

வேகமான தேர்ந்தெடுப்பு
செயப்படுபொருளைக்கொண்டது
செயப்படுபொருளிலாதது
இருபயன்பாடுகளையுங்கொண்டது
வினைப்பகுதி மதிப்பு செய்யப்படவேண்டிய மாற்றம்
தேர்ந்தெடு:
சேர்க்க

செயப்படுபொருளைக்கொண்ட வினைமுற்றுகள்.
