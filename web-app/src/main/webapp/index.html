இந்த செயலியின் நோக்கம் என்ன?

தமிழை புரிந்துகொள்ளுகிற ஒரு தளமென்பொருளை (a platform) உருவாக்குதலே எனது முதற்கட்டமுயற்சி. இது ஒரு குறிப்பிட்ட கடைநிலைப்பயன்பாட்டிற்காகமட்டுமே உருவாக்கப்படும் மென்பொருளல்ல. பலவிதமான பயன்பாடுகளைக்கொண்ட தமிழ்மென்பொருட்களை இதன்மேல் பிற்காலத்திலுருவாக்க இதுவொரு தளமாக அமையவேண்டுமென்பதே இதனுடைய அடிப்படைநோக்கம். எனவே, இந்தச்செயலியில் காணப்படும் பக்கங்கள் கடைநிலைப்பயன்பாட்டாளர்களுக்கானவையல்ல. இந்தச்செயலியை அதனுடைய நோக்கத்தை எட்டவைக்க உதவக்கூடிய தமிழார்வலர்களுக்கானவை அவை.
சில பக்கங்கள் (எ.கா தமிழ்த்திருத்தி ) போன்றவை கடைநிலைப்பயன்பாட்டைக்கொடுத்தாலும் அந்தத்திருத்தி ஒரு சோதனைத்தளமாகவே (test-bed or a reference implementation) பயன்படுகிறது. ஒரு வலுவான தமிழ்த்திருத்தியை செய்யவிரும்புபவர்கள் தாராளமாக செய்யலாம். இந்தத்தளம் (platform) அதற்கு தேவையான ReST APIs களைக்கொடுக்கும். ஓரழகான மற்றும் வலுவான தமிழ்த்திருத்தியை இத்தளங்கொடுக்குஞ்சேவைகளைப்பயன்படுத்தி உருவாக்க ஆவல்கொண்டவர்கள் என்னைத்தொடர்புகொள்ளலாம்.

நீங்கள் பலவழிகளில் இதற்கு உதவலாம்.
  1. வினையடிகளை(root verb) இங்கு சேர்க்கலாம். வினையடிகளிலிருந்து உருவாக்கப்படும் சொற்களை சரிபார்த்து தவறுகளைச்சுட்டலாம்.
  2. வினையடிகளுக்கு இணையான ஆங்கிலச்சொல்லை சேர்க்கலாம். (விரைவில்!)

  6. இந்த முகநூல்-குழுமத்தில் இணைந்து தொழில்நுட்பந்தொடர்பான விடயங்களை பகிரலாம்.
